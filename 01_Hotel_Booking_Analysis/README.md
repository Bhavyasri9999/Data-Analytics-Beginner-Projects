# Hotel Booking Analysis

## Problem statement
Which factors influence hotel booking cancellations?

## Dataset
~119,000 rows, 32 columns. Key fields: hotel type, lead time, cancellation status.

## Key insights
- City hotels have higher cancellation rates than resort hotels
- Longer lead times correlate with more cancellations
- No-deposit bookings cancel more often
- Peak booking months: July and August

## Tools used
Python · Pandas · Matplotlib · Seaborn · Jupyter Notebook
